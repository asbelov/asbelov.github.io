  • Действия (Actions)

  • Action editor: разработка нового действия. Документация по разработке
  • Collector editor: разработка нового сборщика. Документация по разработке
  • Counter settings: создание или редактирование счетчиков. Подробное описание счетчиков
  • Dashboard: средство просмотра и управления событиями системы мониторинга
  • Data browser: средство просмотра исторических данных. Построение графиков
  • Events editor: действие для редактирования событий
  • Import&Export: импорт и экспорт объектов
  • Interface editor: кастомизация пользовательского интерфейса
  • Launcher editor: разработка нового лаунчера. Документация по разработке
  • Media editor: разработка новых средств отправки сообщений. Документация по разработке
  • Object creator: создание новых объектов
  • Object editor: редактирование параметров объекта. Подключение и отключение счетчиков от объекта
  • Objects Information: средство для просмотра свойств и исторических данных объекта
  • Objects clone: клонирование свойств объектов, подключений объектов к счетчикам и взаимодействий
  • Objects interactions: управление взаимодействиями между объектами. Подробное описание взаимодействий
  • Objects properties: создание и редактирование свойств объектов
  • Objects remover: удаление объектов. Удаление вложенных объектов
  • Run cmd script: запуск скрипта или команды cmd
  • Run powershell script: запуск скрипта или команды powershell
  • Schedule viewer: Просмотр расписания
  • Send message: отправка сообщений с помощью communication media
  • Simple test: тестирования работы действий и задач
  • Task maker: создание и редактирование задач. Подробное описание задач
  • Tasks groups editor: редактирование групп для задач
  • Users editor: создание, удаление и редактирование пользователей для доступа к системе
  • Variables: работа с отладочной информацией, полученной для мониторинга процесса вычисления значения счетчика.

  • Сборщики (Collectors)

  • Connect by TCP\IP: проверки доступности сервисов по TCP\IP
  • Constant: возвращает переданный параметр в неизменном виде
  • Data aggregator: обработка данных для нескольких объектов
  • Directory size: вычисляет размер каталогов
  • Disabled events notification: Информирует о скором включении отключенных событий
  • Event generator: формирование событий и оповещений
  • Events aggregator: агрегация произошедших событий
  • File reader: получает данные из файла
  • JavaScript: обработка данных, используя свой JavaScript
  • MSSQL: выполнение запросов к СУБД MSSQL
  • NSLookup: Получает IP адрес по DNS имени
  • Objects discovery: обнаруживает хосты в сетях различными методами
  • Ping: проверка доступности хоста в сети
  • SNMP: получение данных по протоколу SNMP
  • Scheduler: генератор ежедневных событий
  • Task runner: автоматизация процессов - используется для запуска задачи
  • Timer: генератор событий через интервалы времени
  • Zabbix active add: получает данные от Zabbix агентов в активном режиме или в режиме trapper. Порт 10056
  • Zabbix active: получает данные от Zabbix агентов в активном режиме или режиме trapper. Порт 10051
  • Zabbix passive: получает данные от Zabbix агентов в пассивном режиме
  • verPE: Получает версию файла в ОС MS Windows

  • Запуск действий (Launchers)

  • exec: запуск внешних программ без использования оболочки
  • nodeModule: запуск подключаемых модулей nodejs

  • Средства связи (Communication medias)

  • EMail: отправка сообщений по протоколу SMTP
  • QLASHA: Система оповещения через громкую связь
  • SMS: отправка сообщений через СМС сервис провайдеров
  • Telegram: отправить сообщение телеграм боту
