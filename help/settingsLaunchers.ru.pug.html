Описание настроек лаунчеров для действий launchers.json

Описание

Лаунчеры - это модули ALEPIZ, необходимые для запуска действий. Обычно действия запускаются с помощью nodeModule - лаунчера для встраивания действий в программный код ALEPIZ. В этом случае действие должно быть написано на JavaScript и будет выполняться в среде nodejs.

Действия можно выполнять и с помощью других лаунчеров. Например, лаунчер exec позволяет выполнять любые исполняемые файлы ОС. Они могут быть запущены в качестве действий ALEPIZ. Можно создать собственный лаунчер, например, для запуска действий на удаленных серверах с помощью технологии Microsoft WMI или SSH. Информация по созданию лаунчера находится в справке для действия Launcher editor. Информация по каждому лаунчеру так же можно найти в справке по ALEPIZ

Настройки

  • dir - каталог в котором находятся лаунчеры. Обычно launchers.
  • fileName - имя файла лаунчера. Обычно launcher.js
