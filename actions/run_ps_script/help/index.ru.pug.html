Run powershell script

Запускает скрипт или команду powershell

Описание действия

Действие Run powershell script служит для запуска скрипта или команды powershell. Можно указать ключи powershell, рабочий каталог и данные, которые будут переданы на STDIN запускаемой команды или скрипта. В качестве примера запускается скрипт "print data.ps1" из каталога действия Run powershell script.

Возвращаемое значение: Объект {stdout: <stdout>, stderr: <stderr>, exitCode: <exitCode>}

Параметры

  • Arguments for powershell.exe - параметры командной строки, которые будут переданы powershell.exe. Если в параметре присутствует пробел, параметр необходимо указать в кавычках
  • Working dir - рабочий каталог из которого будет запущен powershell.exe. Можно указать относительный путь. В этом случае он будет вычисляться относительно корневого каталога ALEPIZ.
  • STDIN data - данные, которые будут переданы на STDIN скрипта или команды
