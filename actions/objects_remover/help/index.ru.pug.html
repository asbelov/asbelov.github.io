Objects remover

С помощью Objects remover вы можете удалять объекты.
Возвращаемое значение: ничего не возвращает.

Описание полей ввода

Objects list

Для удаления объектов необходимо выбрать один или несколько объектов в системном меню Objects. Выбранные объекты отобразятся в поле Objects list

Delete selected objects with children objects

При установке этого checkbox будут удалены не только выбранные объекты, но и объекты, которые включены в выбранные объекты. Количество вложенных объектов, которые тоже будут удалены - не ограничено. Но одновременно можно удалить не более 10 000 объектов. При использовании Objects remover в задачах автоматизации это значение можно переопределить полем с названием maxObjectsCnt

До выполнения действия появится диалог в котором необходимо еще раз подтвердить намерение удалить выбранные объекты. Только после нажатия на YES объекты будут удалены
