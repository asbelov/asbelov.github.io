Events aggregator

Events aggregator используется для вычисления агрегированного процента для произошедших событий.

Описание Events aggregator

Events aggregator используется для вычисления процента объектов из группы объектов для которых произошло событие. Например, можно отправлять экстренные оповещения администраторам, если определенный процент серверов из кластера, предоставляющего сервис, перестал быть доступным.

Параметры

  • Object group name or objectID of group - имя группы объектов или идентификатор группы объектов, для которой будет производится агрегированная проверка
  • Event counter name or counterID - имя счетчика с событием или его идентификатор, для которого будет производится агрегированная проверка. В качестве сборщика для этого счетчика должен быть обязательно установлен Event generator.

Возвращаемое значение

Процент объектов от общего количества объектов в группе для которых произошло событие.
