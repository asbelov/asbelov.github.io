Timer

Генератор событий через интервалы времени

Описание Timer

Сборщик генерирует события через определенные параметром интервалы времени. Он может использоваться для периодического запуска зависимых от него пассивных сборщиков, если требуется собирать данные через определенные интервалы времени. Обычно при настройке счетчиков на основе этого сборщика, период сохранения истории и трендов устанавливают в 0, чтобы не расходовать зря дисковое пространство.

Параметры

  • Events will be generated at this time interval - Интервал времени в секундах, через который будет генерироваться событие

Возвращаемое значение

Сборщик возвращает текущее время в миллисекундах, начиная с 1.1.1970
