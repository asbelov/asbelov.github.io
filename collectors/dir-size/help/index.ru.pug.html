Directory size

Directory size используется для вычисления размера каталогов.

Описание Directory size

Directory size используется для вычисления размера заданных каталогов. Можно задавать исключения, как в текстовом виде, так и в виде регулярного выражения. Для снижения нагрузки на систему, после вычисления размера каждого файла, выполнение коллектора приостанавливается на заданное время.

Параметры

  • Directory names (comma separated) - список каталогов для подсчета размера (через запятую). Пробелы в начале и конце названия каждого каталога будут удалены.
  • Excluded directories (comma separated) - список каталогов и файлов, которые будут исключаться из подсчета размера (через запятую). Пробелы в начале и конце названия каждого каталога будут удалены.
  • Excluded directories in regExp format - каталоги и файлы для исключения написаны в формате регулярны выражений. Если отключено, то это точные значений.
  • Sleep periodically for reduce load (ms) - Для снижения нагрузки на систему, после вычисления размера каждого файла, выполнение коллектора приостанавливается на заданное время в миллисекундах.
  • Maximum execution time after which a warning will appear in the log (ms) - Максимальное время выполнения вычисления, после которого в лог файл будет добавлено предупреждение.
  • Do not output errors to the log file - не выводить ошибки работы с файлами и каталогами в лог файл. В некоторых случаях отсутствие каталога или файла, с которым работает коллектор может быть нормой и ошибки будут засорять лог файлы. В этом случае имеет смысл отключить вывод ошибок

Возвращаемое значение

Размер каталогов в байтах.
