File reader

Получает данные из файла с помощью регулярного выражения

Описание File reader

Сборщик используется для получения данных из файла, обрабатывая содержимое файла с помощью регулярного выражения

Параметры

  • File name - полный путь до файла
  • Regular expression - регулярное выражение для получения необходимого параметра. Для обработки содержимого файла используется функция JavaScript replace(/regExp/igms, "$1");
  • Don't log errors - не писать в лог ошибки. Например, в некоторых случаях файла может не существовать и это не является ошибкой. В этом случае можно включить эту опцию чтобы не писать в лог лишнюю информацию.

Например, для содержимого файла:

[GENERAL]
MAXCONN_REPLY=1
MAXCONN=30
MAXCONN_GLOBAL=30

Если используется регулярное выражение "^.+?MaxConn_Global *= *(\d+).*?$", результат будет равен 30

Возвращаемое значение

Сборщик возвращает данные, которые вернет регулярное выражение при обработке содержимого файла.
