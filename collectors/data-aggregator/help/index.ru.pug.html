Data aggregator

Data aggregator используется обработки данных, полученных для счетчика, подключенного к нескольким объектам.

Описание Data aggregator

Data aggregator используется для обработки данных, полученных для определенного счетчика, который подключен к нескольким объектам. Список объектов можно задать через запятую или через подстановки (wildcards) SQL Lite. Например, сборщик позволит посчитать суммарную утилизацию CPU для кластера, либо средний расход памяти для процессов определенного типа.

Параметры

  • Object names (comma separated in SQL Like format) - Список объектов через запятую. В именах объектов можно использовать подстановки SQL Like формата ("_" - любой один символ, "%" - ноль или больше любых символов). Начальные и конечные пробелы для каждого объекта будут удалены.
  • counterName - имя общего для выбранных объектов счетчика, для которого будут получены и обработаны исторические данные. Если указанный счетчик не подключен к части объектов, эти объекты будут пропущены. Начальные и конечные пробелы будут удалены.
  • Function - функция, используемая для обработки данных:
    • sum - полученные данные складываются
    • avg - вычисляется среднее арифметическое для полученных данных
    • min value - возвращается минимальное значение из полученных данных
    • max value - возвращается максимальное значение из полученных данных

Возвращаемое значение

Значение, полученное в результате обработки данных
