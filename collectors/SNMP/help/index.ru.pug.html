SNMP

Выполняет запросы по протоколу SNMP и возвращает полученные значения

Описание SNMP

Сборщик выполняет запросы к сетевому хосту по протоколу SNMP и возвращает полученное значение.

Параметры

  • IP address or host name - IPv4 или IPv6 адрес или имя хоста, для получения данных
  • SNMP community - SNMP комьюнити, обычно это public
  • SNMP OID - OID, по которому планируется получать данные. Для проверки можно использовать 1.3.6.1.2.1.1.1.0, возвращающий информацию о системе
  • SNMP TCP port - TCP порт, обычно 161
  • SNMP protocol version (1 or 2) - Версия SNMP протокола: 1 или 2
  • Number of SNMP Request Attempts - количество попыток запросов, если хост не вернул данные
  • SNMP request timeout (sec) - таймаут для SNMP запроса, по истечении которого ожидание данных будет прервано. По умолчанию 5 секунд

Возвращаемое значение

Сборщик возвращает полученные при опросе значения
