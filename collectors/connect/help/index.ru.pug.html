Connect by TCP\IP

Сборщик используется для проверки доступности сервисов по протоколу TCP\IP

Описание Connect by TCP\IP

С помощью Connect by TCP\IP можно проверять доступность сервисов по протоколу TCP\IP, подключаясь к ним на указанные в параметрах коллектора IP адреса (или имена хостов) и определенный порт.

Параметры

  • Comma separated IPv4 or IPv6 addresses or host names - перечисленные через запятую адреса IPv4 или IPv6 или имена хостов, на которых требуется проверять работу сервиса
  • TCP port - TCP порт, который использует проверяемый сервис
  • Source IPv4 or IPv6 address or host name - IPv4 или IPv6 адрес локального интерфейса, с которого необходимо устанавливать соединение. Параметр является необязательным.
  • Socket timeout in millisecond - Таймаут в миллисекундах, в течение которого коллектор будет пытаться установить соединение. Если в течение этого времени соединение установить не получится, будет считаться, что хост недоступен. По умолчанию 10000. Параметр является необязательным.

Возвращаемое значение

Перечисленные через запятую хосты из параметра hosts, к которым удалось подключиться на порт port. Если подключиться никуда не удалось, вернется пустая строка "". Если хост был указан один, и подключиться к нему удалось, вернется адрес или название хоста в том виде, как он был указан в параметре hosts
