Constant

возвращает переданный параметр в неизменном виде

Описание Constant

Сборщик возвращает переданный параметр в неизменном виде. Он может быть использован для обработки данных, полученных с помощью других сборщиков, встроенными функциями ALEPIZ. Для этого в качестве параметра ему необходимо передать переменную, содержащую предварительно обработанное значение.

Параметры

  • This value will be returned by the collector - параметр, который будет возвращен сборщиком

Возвращаемое значение

Сборщик возвращает переданный параметр в неизменном виде
