Javascript

Сборщик используется для преобразования данных с помощью Javascript

Описание Javascript

Вы можете написать собственный JavaScript код, используя переменные счетчика для обработки и преобразования данных.

Для использования переменных, необходимо включить их в код JavaScript в стандартной для переменных ALEPIZ форме %:<Variable>:%. Перед запуском сборщика препроцессор заменит их имена на значения. Если переменная возвращает строку, ее нужно поместить в кавычки.

Возвращаемое функцией значение будет результатом, возвращаемым счетчиком.

myFunc();

function myFunc() {
    var objectName = "%:OBJECT_NAME:%";
    return 'NEW_' + objectName;
}

Параметры

  • javascript - код на JavaScript, который будет запущен при вызове сборщика

Возвращаемое значение

Сборщик возвращает результат работы функции JavaScript
